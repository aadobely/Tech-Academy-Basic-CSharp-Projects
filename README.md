# C# and .NET Framework Projects

Collection of projects I built while learning C# and the .NET Framework at Academy of Learning Career College.

## Projects:

### [Package Express](./PackageExpress)
Console application for calculating shipping quotes based on package weight and dimensions. Validates input and calculates costs using business logic.

**Skills:** Console apps, input validation, conditional logic, arithmetic operations

---

### [Void Method Assignment](./VoidMethodAssignment)
Demonstrates understanding of void methods, method parameters, and named arguments in C#.

**Skills:** Methods, parameters, named arguments, void methods

---

### [Interface Assignment](./InterfaceAssignment)
Implementation of interfaces and polymorphism. Creates an IQuittable interface and implements it in an Employee class.

**Skills:** Interfaces, polymorphism, inheritance, OOP principles

---

### [Operator Overload Assignment](./OperatorOverloadAssignment)
Demonstrates operator overloading by creating custom equality operators for an Employee class that compares based on ID.

**Skills:** Operator overloading, comparison operators, class design

---
